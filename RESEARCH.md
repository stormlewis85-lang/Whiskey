## Prior Research (Pre-Agent)

### Instagram Graph API for Store Monitoring
- Instagram Graph API supports reading posts from accounts we follow
- Follow cap: 7,500 accounts (sufficient runway for MVP)
- API rate limits: ~200 calls/hour for basic endpoints
- Polling frequency: every 15-30 minutes is sufficient
- No TOS violations when reading our own feed
- Business account required for API access

### Competitive Landscape
- **Distiller:** Collection tracking + reviews. No store alerts, no clubs, no AI.
- **Whiskey Suggest:** Collection + reviews. No store alerts, no clubs, no AI.
- **Untappd (beer):** Collection + reviews + limited store following. No whiskey-specific features.
- **MyWhiskeyPedia differentiator:** Hunt utility + social depth + AI guidance (Rick House) in one platform.
